## Adaptive UI using AI/HCI Project

### Project Title
Adaptive User Interface Adaptive Human-Computer Interaction with Gaussian Naive Bayes

### Project Description
A dynamic quiz system based on Gaussian Naive Bayes was created to adapt the question difficulty according to performance indicators like score and errors. Tkinter was used to develop the graphical interface with visual effects like level indicators and background color dynamically changing to increase the interaction and ease of use.

### Objectives
Building a machine learning-based quiz.

Gaussian Naive Bayes prediction of difficulty levels.

Change in difficulty of questions in response to performance.

Adjusting interface features like colour and level display.

Improved user interaction by being adaptive.

### Features
Dynamic difficulty adjustment.

Real-time performance tracking.

Color-based interface adaptation.

Graphical user interface that is interactive.

Tracking of scores and errors.

Exit confirmation option.

### Technologies Used
Python

Tkinter

NumPy

Scikit-learn

Gaussian Naive Bayes

### System Functionality
Performance metrics such as score and mistakes were used as input.

Gaussian Naive Bayes was used to predict the difficulty level.

The questions were chosen according to their difficulty.

Color of interface varied with level of difficulty.

Response feedback was given.
